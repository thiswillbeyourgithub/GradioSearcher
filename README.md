# gradioSearch

A CLI tool for searching FAISS vector databases with a Gradio GUI interface.

**Work in progress.**

## Installation

```bash
git clone https://github.com/thiswillbeyourgithub/gradioSearch.git
cd gradioSearch
uv pip install -e .
```

## Usage

```bash
python -m gradioSearch --help
```
